# Portfolio Website

This portfolio website is built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main portfolio page
- `styles.css` — responsive styling and layout
- `script.js` — mobile navigation toggle and active section highlighting

## How to use

1. Open `index.html` in your browser.
2. If you want to host it, upload the folder to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).
3. Update the content in `index.html` as needed for your name, projects, and contact details.

## Notes

- The site is responsive and designed for desktop and mobile screens.
- To make changes, edit the HTML and CSS as desired.
